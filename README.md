# I Thought of That

A GitHub Pages website for the song **I Thought of That**, written for **Christine Goodwin**.

## Files

- `index.html` — main webpage
- `audio/I Thought of That.mp3` — song audio
- `video/I Thought of That.mp4` — song video

## GitHub Pages Setup

1. Create a repository named `i-thought-of-that`
2. Upload all files from this folder
3. In GitHub, open **Settings > Pages**
4. Under **Build and deployment**, choose **Deploy from a branch**
5. Select the `main` branch and `/ (root)` folder
6. Save and wait for GitHub Pages to publish

Your site will then be live on GitHub Pages.
